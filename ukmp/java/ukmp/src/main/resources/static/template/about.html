Welcome to the Flax UKMP application, providing search and analysis of tweets posted by UK members of Parliament.

Credit where it's due...

The UKMP application is built using a number of Open Source technologies, including:

  • Apache Solr, which provides the search engine back-end for storing and searching over the tweet data.
  • DropWizard, which provides the web back-end framework, allowing the front-end to communicate with Solr, and the indexing code to interact with the Stanford NLP API.
  • Stanford NLP for entity extraction and sentiment analysis.
  • AngularJS, providing the framework for the web application.
  • Twitter Bootstrap for the layout, and including the Glyphicons set used in the search/browse page.
